Links con HTML y CSS


Para modificar el estilo de los links segun su estado podemos usar CSS3
a:link es el estado default
a:visited es para cuando ya lo visitamos
a:hover es para cuando nos paramos sobre el link
a:active es para cuando estamos haciendo click
con el attribute text-decoration puedo quitar el subrayado a los link para cuando esten o no visitados hovereados
