Elementos inline y block

        Elementos inline , hablamos de elementos que se aplican dentro de la misma linea en la que estamos trabajando
                - Ejemplo <span> <h1> 
        Elementos block, hablamos de elementos que siempre se inician en una nueva linea:
                - Ejemplo <div>
