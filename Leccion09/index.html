Formas de aplicar estilos con CSS

Se puede aplicar estilos CSS con:

  • Estilos en el tag HTML con el attributo de todo tar HTML llamado style
  • Usando la etiqueta <style> en el head del documento HTML y aplicando los estilos alli
  • creando un archivo *.css e importandolo mediante el tag HTML <link>

En ese mismo orden es la prioridad con que se aplican los estilo
La prioridad de las Cascadas puede sobreescribirse utilizando !important


ID

Para aplicar estilos podemos utilizar el selector ID podemos definir estilos para elementos unicos. Para hacerlo definimos en CSS #idname utilizando el selector de id #


Class

Para aplicar estilos tambien podemos utilizar el selector Class, para definir estilos para elementos reiterativos especificos
Para hacerlo definimos en CSS .classname utilizando el selector de class .
