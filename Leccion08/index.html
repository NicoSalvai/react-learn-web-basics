Formularios en HTML

Para crear un formulario usamos la etiqueta <form>
Para poder capturar informacion agregamos elementos del tipo <input>
En el tenemos que agregar el atributo type para indicar si es numero fecha texto pwd etc
Al definir inputs siempre deberemos definir un id y un atributo name
Para poder enviar unformacion deberiamos tener siempre un input de type="submit"
Para indicar que es cada campo se usan <label> con el atributo for valuado en el id del campo
Algunos de los types disponibles son:

En el caso de radio y checkbox id = - name = Nombre del parametro que envio al BE (esto los agrupa para que sean autoexcluyentes) value = valor que se envia al server
En el caso de los select podemos usar multiple para que sea multiselect y selected para poner la opcion por default
En el caso de querer manejar mucho texto se usa la tag <textarea>

Tambien podemos poner un boton de reset con el type reset

Validaciones


Podemos agregar validaciones sobre los formularios:
El atribbuto required="true" del tag input hace que no se pueda enviar si ese campo esta vacio

Se puede quitar el autocompletado con el atributo autocomplete seteado el false

Tambien podemos hacer campos que se autocompleten y no puedan ser modificados

Metodos HTTP y acciones

Por defecto el metodo de todo formulario es un GET
puedo cambiar el metodo de un formulario con el atributo method="POST" del tag <form>
podemos usar el atributo action del tag form para indicar la URL destino de la peticion
tambien tenemos el atributo target para ver donde queremos mostrar la respuesta a la peticion

Fieldset y Legend

fieldset es para encuadrar/encasillar un form
legend es para dar un titulo al form en ese mismo encuadre
