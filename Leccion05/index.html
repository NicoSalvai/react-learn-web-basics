Tablas en HTML

con la etiqueta <table> declaramos una tabla
Luego definimos cabecero y renglones:
Usamos la etiqueta <tr> para definir un nuevo table row osea un renglon
Usamos la etiqueta <th> dentro de una row para definir un nuevo table header osea un cabecero
Usamos la etiqueta <td> dentro de una row para definir un nuevo table data osea una celda
Las tablas tienen la property border que si lo valuamos en 1 crea un borde por defecto
La etiqueta <tbody> va dentro de table y es el cuerpo de la tabla pero podemos omitirla
