Parrafos

Los parrafos no tienen saltos de linea dentro de un mismo <p>

Si quiero hacer salto
de linea entre parrafos uso <br>

Para hacer una linea horizontal como la que esta arriba usamos <hr>

Imagenes

Usamos la etiqueta <img> en la property src va el link o path a la imagen
tambien podemos usar la property alt para indicar un texto que se muestre si la imagen no esta disponible
tambien podemos usar la property width y height para especificar el tama単o

Manejo de attributos

por ejemplo en casi cualquier elemento tenemos el attributo title para ver un titulo al pararnos sobre el
tambien podemos usar style para cambiar el color del texto o titulo o cualquier estilo css
